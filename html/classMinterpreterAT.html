Detailed Description

Class used to build or parse AT messages (this class derives UWMinterpreter); currently, this class implements methods to build/parse:

  • simple configuration messages (e.g., setting of the modem ID)
  • messages involved in the transmission of an instant message (see AT*SENSIM command) NOTE: for a detailed description of the messages build and parse in this class, please refer to the S2C Underwater Acoustic Modem Guide. Contact EvoLogics GmbH, Ackerstr. 76 d-13355 Berlin for more infos (email: suppo.nosp@m.rt@e.nosp@m.volog.nosp@m.ics..nosp@m.de).
