Detailed Description

Class that implements the interface between ns2/NS-Miracle and the S2C EvoLogics modem (it derives UWMPhy_modem).

NOTE: For tcl-user, set node($id) [new "UW/MPhy_modem/S2C" "path to the device"] Example: in case of a modem to be connected via TCP/IP socket "path to the device" would be the application port opened via ssh, e.g., 9200

◆ check_modem()

modem_state_t MS2C_EvoLogics::check_modem ( )
virtual

Modem checker.

This method is at the core of the "check-modem" process. It is called periodically by the timer object linked to this UWMPhy_modem object (see UWMPhy_modem::checkTmr_); its due is to verify if something has been received or is going to be received from the channel.

Returns
modemStatus, a flag on the status of the modem, see UWMdriver for a description of the driver state machine.

Implements UWMPhy_modem.
