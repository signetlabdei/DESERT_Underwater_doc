Detailed Description

Class inherited the method and variable of uwUFetch_NODE_timer that handle the timer of DATA packets.

HN set up a time interval within It want to receive all DATA packets from a specific SN. After the end of this time interal HN start a new transmission of POLL or CBEACON or wait a TRIGGER packet from AUV.
