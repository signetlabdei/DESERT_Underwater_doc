Detailed Description

Class inherited the method and variable of uwUFetch_NODE_timer that handle the timer of TRIGGER packets. HN set an interval time within It wait a TRIGGER packet from AUV. When this interval time expire, if HN has not received any TRIGGER It start the communication with SN, otherwhise It start the communication with AUV.
