Detailed Description

Class inherited the method and variable of uwUFetch_NODE_timer that handle the timer of CBEACON packets.

SN set a time interval within it want to receive a CBEACON packet from a specific HN. If this timer expired the SN return in Idle state and wait a new BEACON packet from any HN
