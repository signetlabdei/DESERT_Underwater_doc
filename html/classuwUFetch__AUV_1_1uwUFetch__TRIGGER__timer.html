Detailed Description

Class inherited the method and variable of uwUFetch_timer that handle the timer of TRIGGER packets.

Before to transmit in broadcast to the HNs a TRIGGER packet, the AUV node wait a finite interval time, after that start with the creation and transmission of TRIGGER packet.
