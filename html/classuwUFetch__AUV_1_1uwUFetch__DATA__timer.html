Detailed Description

Class inherited the method and variable of uwUFetch_timer that handle the timer of DATA packets.

After the transmission of CTS packet, the AUV node enter in a waiting state in which It wait a sequence of DATA packets requests from HN. The uwUFetch_DATA_timer schedule this waiting interval time
