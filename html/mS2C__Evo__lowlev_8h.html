Detailed Description

Header of the main class that implements the general interface between ns2/NS-Miracle and the S2C EvoLogics modem using low level firmware. Only modems on which the firmware was made available can be operated through this type of drivers. For information: www.evologics.de, EvoLogics GmbH.
