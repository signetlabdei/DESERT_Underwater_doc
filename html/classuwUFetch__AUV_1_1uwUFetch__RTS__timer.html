Detailed Description

Class inherited the method and variable of uwUFetch_timer that handle the timer of RTS packets. When the TRIGGGER packet is transmitted, the AUV node wait for a finite interval time a RTS packet from the HN. The uwUFetch_RTS_timer handle this waiting interval time.
