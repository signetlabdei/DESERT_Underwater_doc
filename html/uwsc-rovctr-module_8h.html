Provides the definition of the class UwSCROVCtr. UwSCROVCTR receives cross layer messages containing the next waypoint that has to be sent to the associated node follower. In addition it receives monitoring packets containing the current node folllower position. SCROVCtr stands for Swarm control ROV Controller.
