Detailed Description

Class inherited the method and variable of uwUFetch_NODE_timer that handle the timer of POLL packets.

SN set up a time interval within It's enabled to receive a POLL packet from a specific HN. If in this time interval the HN node doesn't transmit the POLL packet the SN return in IDLE state or It start to wait the successive CBEACON packet from that HN.
